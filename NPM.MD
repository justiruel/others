### install locally
```
npm install --save <library name>
```
### install globally
```
npm install -g <library name>
```
### list globally installed library
```
npm list -g --depth=0
```
check version
```
npm --version
```

### eksekusi hasil npm install
```
var _ = require(<library name>);<br/>
contoh :
var _ = require('lodash');
```
